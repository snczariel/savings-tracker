Create your Google Sheet
Make a new Google Sheet. Rename the first tab to Entries. Add these headers in row 1:
Date  |  Fund  |  Account  |  Amount  |  Who  |  Remarks  |  Timestamp

Add the Apps Script
In your Sheet: Extensions → Apps Script. Replace all code with the script. Then: Deploy → New deployment → Web app. Set "Who has access" to Anyone. Copy the URL.
